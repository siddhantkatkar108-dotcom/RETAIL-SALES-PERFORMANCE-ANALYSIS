# Retail Sales Performance Analysis Dashboard

### Project Overview
This project analyzes retail sales data using Excel, SQL, and Power BI. The dashboard provides insights into sales performance, profit trends, customer behavior, product performance, and business growth.

### Objectives
- Analyze overall sales performance.
- Identify top-performing products and categories.
- Track sales and profit trends.
- Evaluate customer purchasing behavior.
- Generate actionable business insights.

### Tools Used
- Microsoft Excel
- SQL
- Power BI

### Dataset Features
- Order ID
- Order Date
- Product Category
- Product Name
- Customer Segment
- Sales
- Profit
- Quantity
- Region

### Dashboard KPIs
- Total Sales
- Total Profit
- Total Orders
- Average Order Value
- Top-Selling Products
- Category Performance
- Regional Sales Analysis
- Monthly Sales Trends

### Process
1. Data Cleaning in Excel
2. Data Transformation
3. SQL Analysis
4. Power BI Data Modeling
5. DAX Measure Creation
6. Dashboard Development
7. Insight Generation

### Key Insights
- Identified top-performing products contributing to overall revenue.
- Analyzed category-wise sales and profit performance.
- Evaluated regional sales trends and business growth.
- Discovered customer purchasing patterns and sales opportunities.
